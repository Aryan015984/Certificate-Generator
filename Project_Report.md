# 1. Introduction to Project

## 1.1 Project Overview

The Certificate Generator is a comprehensive web-based application designed to provide users with an intuitive platform for generating professional certificates. The system offers a complete workflow from user authentication to certificate creation and printing.

## 1.2 Objectives

The primary objectives of this project are:

- **User Authentication:** Implement secure login and signup functionality
- **Course Management:** Provide access to various educational courses
- **Certificate Generation:** Create professional-looking certificates dynamically
- **Responsive Design:** Ensure compatibility across different devices
- **Modern UI/UX:** Deliver an attractive and user-friendly interface

## 1.3 Scope and Limitations

### Scope:
- User registration and login system
- Course selection interface
- Dynamic certificate generation
- Print functionality for certificates
- Responsive web design

### Limitations:
- No backend database integration
- Client-side data storage only
- No email functionality for certificate delivery
- Limited to predefined course options

## Appendix A: Source Code Excerpts

### Certificate Generation JavaScript
```javascript
document.querySelector('#certPage .btn').addEventListener('click', function() {
  const name = document.getElementById('recipientName').value.trim();
  const course = document.getElementById('courseSelect').value;
  if(name) document.getElementById('certName').textContent = name;
  if(course) document.getElementById('certCourse').textContent = course;
});
```

### CSS Glassmorphism Effect
```css
.form-box {
  background: rgba(255,255,255,0.1);
  backdrop-filter: blur(10px);

}
```
